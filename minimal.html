# What is a minimal website

It is a website without bells and whistle that can still serve you .

This site can be loaded in you browser with solely 1.2KB of data
transferred or 0B if you run IPFS locally.  And it took only [11s][1]
to deploy the site "world wide" on [Netlify][2].
